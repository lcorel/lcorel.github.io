Hexo博客Next主题添加友链页面

发表于 2020-05-11 分类于 博客 阅读次数： Valine：
本文字数： 3.2k 阅读时长 ≈ 3 分钟

配置菜单及汉化

主题配置文件hexo/themes/next/_config.xml 中menu下添加： 

links: /links/ || fa fa-link

主题语言配置文件 hexo/themes/next/languages/zh-Hans.yml文件中menu下增加中文描述： 

links: 友链

添加友链页面

hexo new page links

修改友链页面内容

在 hexo/blog/source/links 目录下index.md

----
title: 友情链接
date: 2020-05-11 15:42:13
type: "links"
---

---

### 申请要求：

1、内容持续更新且可以稳定访问
2、网页整洁无繁杂广告推广
3、博客主页被百度或谷歌等搜索引擎收录
4、头像能够快速加载
5、拥有独立域名


### 友链声明：

1、本站会主动保存您的 HTTPS 形式的头像图片链接
2、本站会定期清理无法访问的友链，如果更换了链接信息请至评论区留言，谢谢合作！
3、本站会定期查看双方是否互为友链，如果取消本站友链，本站也会将您的友链移除

### 申请方式：

先将本站的友链添加到您的友链，相关信息如下
然后按照以下格式在本站留言区留言，待博主为您添上友链

>名称：ZZONN
>
>主页链接：https://blog.zzonn.com/ 
>
>头像链接：https://blog.zzonn.com/images/avatar2.gif
>
>说明信息：提高生产力，促进中国特色社会主义的发展，维护世界和平，放飞自我

添加友链布局文件

在 hexo/themes/next/layout 目录下新建 links.njk文件，写入以下代码


{% block content %}
  {######################}
  {### LINKS BLOCK ###}
  {######################}

    

        
        


             
👨‍🎓 跟着大佬走，成为小大佬

            

                    {% for link in theme.defaultlinks %}

                    

                        
                        

                        
 {{ link.nickname }}  关注

                        
{{ link.info }}

                        

                    


                {% endfor %}

            


            
🍭 五湖四海的朋友们


             

                    {% for link in theme.friendslinks %}

                    

                        
                        

                        
 {{ link.nickname }}  关注

                        
{{ link.info }}

                        

                    


                {% endfor %}

            


            {{ page.content }}
            

        


  {##########################}
  {### END LINKS BLOCK ###}
  {##########################}
{% endblock %}

修改page布局文件

同样在 hexo/themes/next/layout 目录下

  1. 找到代码

      {%- elif page.type === 'schedule' and not page.title %}
    {{- __('title.schedule') + page_title_suffix }}

  2. 在后面加入代码

    {%- elif page.type === 'links' and not page.title %}
    {{- __('title.links') + page_title_suffix }}

  3. 找到代码

    {% elif page.type === 'schedule' %}
          <div class="event-list">
          </div>
          {% include '_scripts/pages/schedule.njk' %}

  4. 在后面加入代码(引入刚刚新建的links布局文件)

    {% elif page.type === 'links' %}
          {% include 'links.njk' %}

修改主题配置文件

主题配置文件hexo/themes/next/_config.xml 末尾添加： 

# 友情链接
defaultlinks:
  - nickname: ZZONN     # 昵称
      site: https://blog.zzonn.com/ #友链地址
    avatar: https://blog.zzonn.com/images/avatar2.gif    # 头像地址
    info: 提高生产力，促进中国特色社会主义的发展，维护世界和平，放飞自我

friendslinks:
  - nickname: ZZONN     # 昵称
      site: https://blog.zzonn.com/ #友链地址
    avatar: https://blog.zzonn.com/images/avatar2.gif    # 头像地址
    info: 提高生产力，促进中国特色社会主义的发展，维护世界和平，放飞自我

defaultlinks:呼应的是link.swig文件中 👨‍🎓 跟着大佬走，成为小大佬段落，此处链接写大佬的博客；
friendslinks:呼应的是link.swig文件中 🍭 五湖四海的朋友们段落，此处链接写朋友的博客。
此处内容可以根据自己需要自行修改
